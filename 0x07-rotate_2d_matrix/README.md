# Rotate 2D Matrix

A Python function to rotate an `n x n` 2D matrix 90 degrees clockwise.

## Usage

Import the function and pass your 2D matrix as an argument:

```python
from rotate_matrix import rotate_2d_matrix

matrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]

rotate_2d_matrix(matrix)
print(matrix)
